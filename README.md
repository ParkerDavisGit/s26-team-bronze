# s26-team-bronze

A pantry management application with food recall monitoring.

## Prerequisites

- Node.js (v18 or higher)
- npm
- Git

## Setup Instructions

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd s26-team-bronze
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Setup the database**
   ```bash
   npx prisma generate
   npx prisma db push
   ```

4. **Start the application**
   ```bash
   npm start
   ```

5. **Access the application**
   - Open your browser to `http://localhost:3000`

## Environment Variables

Create a `.env` file in the root directory if needed for configuration (check with team for any required environment variables).

## Database

This application uses SQLite with Prisma ORM. The database file will be created automatically when you run the setup commands.
